说明：

 

用户选择图片动态后，跳转到图片选择界面，包含拍照按钮。选择好后，进入发布图片动态页面，最多可加9张图片，图片可删除，增加，动态内容是可选的，最多输入300个字符，点击发布按钮时，判断是否有图片，如果没有图片，则提示“请添加图片”，如果所有信息都符合规范，则提示“发布成功”，并跳转至动态列表页面

 

 

 

 

 

 

 

 

说明：

用户选择视频动态后，跳转到视频录制页面，按住黄色按钮不放进行录制，松开时，会出现删除和完成按钮，删除则从头开始录制，完成则进入发视频动态页面，视频可删除，增加，动态内容是可选的，最多输入300个字符，点击发布按钮时，判断是否有视频，如果没有视频，则提示“请添加视频”，如果所有信息都符合规范，弹出上传等待框，如果上传成功，则提示“发布成功”，并跳转至动态列表页面，如果失败，则弹窗消失，并提示“上传失败，请重试”，并停留该页面。
