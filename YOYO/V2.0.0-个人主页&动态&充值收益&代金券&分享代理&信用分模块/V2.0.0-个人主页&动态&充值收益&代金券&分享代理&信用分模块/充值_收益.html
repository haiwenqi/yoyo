交互说明：

1.目前收益来自于礼物和付费约；

 

2、收益分成的提现流程：

用户每周只能够申请一次提现，不低于1000元；

用户申请后，会得到一个提现码，然后关注官方微信公众号，和客服联系，客服确认真实性后进行转钱（后期会开发自动化提现功能）

 

3、提取规则

• 每周只可提现一次，每次不低于1000元；

• 申请提现后请关注我们的官方微信公众号，将提现码发给客服；

• 按照国家法律法规，提现金额将收取20%的劳务所得税；

• 提现金额将在24小时内以微信红包形式发到您的微信账户中；

• 最终解释权归官方所有；

•

• 注意：

提现的分成和收税比例计算都是后台计算的

 

 

点击“申请提现”按钮交互

• 如没有填写数字，则提示“请输入提现金额”；

• 如填写其它字符或者带小数的，则提示“请输入正确的整数金额”；

• 如填写小于1000元的金额，则提示“请输入不少于1000元的金额”；

• 如用户没有没有认证，则弹出认证提示框；

• 条件都满足后和后台进行交互，交互完毕后进入提现记录页面；
