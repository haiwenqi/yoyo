聊天页面----说明：

 

当两个用户聊天时，且是异性和不是结伴树关系，则显示匹配度信息，如左图所示；

 

结伴树匹配：

• 匹配度算法第一版本是随机从库中取得，后期迭代中会根据用户的信息进行匹配。

 

IM服务：

• 采用IM服务SDK，具体由技术评估；

• IM服务中保留语音消息、相册、拍照功能；

• 新增发红包、送礼物功能；

 

 

发红包弹窗----说明：

 

用户点击红包图标，弹出红包弹窗，如左图所示；

 

发红包规则：

• 第一版只发金币红包；

• 最少1金币，上不封顶，必须是整数；

• 点击按钮进行发送的时候要判断2个情况，

• 1、是否有网络，无网络则提示“请检查网络连接”

• 2、账户中的金币余额是否充足，不足抵扣则提示“账户余额不足，请充值”；

 

 

发礼物弹窗----说明：
