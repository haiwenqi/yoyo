钻余额页面----说明；

 

交互说明：

1.页面展示该用户当前剩余的钻数；

 

2.用户充值模式是5+1，数据从后台取，具体规则请参考后台管理中的钻充值管理；

 

3.用户首次充值第一个选项是1元，第二次充值即回复原来价格；

 

4、支付方式除了微信和支付宝，还有余额支付，余额是从用户开红包得来的（展示余额数字，保留小数点后两位），默认选择方式从后台取，取不到则以支付宝为默认方式；

 

5、当选择余额支付的时候，点击“立即支付”判断是否足够支付，如果不足以，则提示“余额不足，请选择其它方式支付！”

 

 

 

 

现金页面----说明；

 

交互说明：

1.目前钱的来源只有红包；

 

2、现金的提现流程：

用户每周只能够申请一次提现，不低于50元；

用户申请后，会得到一个提现码，然后关注官方微信公众号，和客服联系，客服确认真实性后进行转钱（后期会开发自动化提现功能）

 

3、提取规则

• 每周只可提现一次，每次不低于50元；

• 申请提现后请关注我们的官方微信公众号，将提现码发给客服；

• 按照国家法律法规，提现金额将收取20%的劳务所得税；

• 提现金额将在24小时内以微信红包形式发到您的微信账户中；

• 最终解释权归官方所有；

 

 

点击“申请提现”按钮交互

• 如没有填写数字，则提示“请输入提现金额”；

• 如填写其它字符或者带小数的，则提示“请输入正确的整数金额”；

• 如填写小于50元的金额，则提示“请输入不少于50元的金额”；

• 点击后进入提现记录页面；
